# 🐳 Car Data Dockerized ETL Pipeline

This project demonstrates a complete **Data Engineering** workflow using Python and Docker. It automates the process of cleaning raw car dataset records for analysis.

## 🛠️ Features
- **Containerization:** The entire ETL logic is packaged using Docker for consistent execution.
- **Data Cleaning:** Uses `pandas` to handle missing values and filter records (Cars newer than 2015).
- **Docker Volumes:** Implements volume mounting to persist data between the host machine and the container.
- **Portability:** Can be run on any system with Docker installed.

## 📁 Project Structure
- `cleaner.py`: The Python script containing the ETL logic.
- `Dockerfile`: Instructions to build the Docker image.
- `car details v4.csv`: The raw input data.
- `cleaned_car_data.csv`: The final output (generated by the container).
